Dragon Ball Xenoverse Tag Team - PPSSPP MOD

This is a modded version of Dragon Ball Z: Tenkaichi Tag Team featuring a complete overhaul with Dragon Ball Xenoverse 2 UI and enhanced textures. Play on your Android using the PPSSPP emulator. No root required.

How to Install

  1. Download the ISO and Texture files below.
  2. Use ZArchiver to extract both downloaded files.
  3. Move the .iso file into your phone's PSP > GAME folder.
  4. Move the ULUS10537 texture folder into the PSP > TEXTURES directory.
  5. Open the PPSSPP emulator, find the game, and start playing!
